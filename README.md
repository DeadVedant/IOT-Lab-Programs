## 📌 Description

This repository contains a collection of Internet of Things (IoT) programs and practical implementations developed as part of college laboratory coursework under SPPU (Savitribai Phule Pune University).

The code is primarily written in C/C++ and is designed for microcontroller-based development using platforms such as Arduino UNO and other commonly used IoT development boards. It focuses on fundamental concepts of embedded systems, sensor interfacing, and real-time hardware interaction.

All programs are intended to be compiled and uploaded using the Arduino IDE or similar development environments compatible with microcontroller programming.

This repository serves as a structured reference for IoT lab experiments, helping students understand practical implementations, improve coding skills, and prepare effectively for university lab work and evaluations.

---

## ⚙️ Setup Instructions
1. Install Arduino IDE
2. Connect your Arduino board
3. Upload the code from respective folders
4. Open Serial Monitor (if required)
